# Z_ABAPGIT_PULL_MCP_SHORTCUT
The ABAP package `Z_ABAPGIT_PULL_MCP_SHORTCUT` contains a minimalistic report and transaction that encapsulate the [abapGit API](https://docs.abapgit.org/development-guide/api/api.html).
They allow MCP agents to pull code from git w/o navigating through the entire abapGit UI.
Having abapGit [installed](https://docs.abapgit.org/user-guide/getting-started/install.html) is a prerequisite for using this package.

Both the report and package name is `Z_ABAPGIT_PULL_MCP_SHORTCUT` but the Transaction Code (T-Code) ist `Z_ABAPGIT_PULL_MCP`;
I wasn't aware the number of characters is that limited.
